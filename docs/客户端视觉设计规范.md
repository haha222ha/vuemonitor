# XHS365 客户端视觉设计规范

> 日期：2026-05-14 | 版本：V2.0 | 基于：产品重构方案V2.0 + UI/UX方案V2.0

---

## 一、设计语言

### 1.1 品牌定位

**XHS365 = 电商情报终端**

视觉风格介于"专业金融终端"和"现代SaaS工具"之间——比通用后台更有品牌感，比消费级App更有专业度。

### 1.2 设计关键词

**深邃 · 洞察 · 信赖**

- **深邃**：深色侧边栏+靛蓝主色，传递"深度洞察"的品牌感
- **洞察**：数据可视化优先，让趋势/排名/异动一目了然
- **信赖**：一致的视觉系统、专业的排版、克制的动效

### 1.3 视觉风格参考

| 维度 | 参考对象 | 借鉴点 |
|-----|---------|-------|
| 侧边栏 | Notion / Linear | 深色+分组+折叠 |
| 数据卡片 | Stripe Dashboard | 大数字+小标签+趋势 |
| 告警系统 | Datadog / Grafana | 严重级别颜色编码 |
| AI展示 | ChatGPT / Claude | 结构化分段+代码块 |
| 商品卡片 | Shopify Admin | 双列信息+快捷操作 |

### 2.8 暖色辅助色（电商场景增强）

Indigo主色传递"专业、科技"，但电商卖家更关注"机会、收益"。
暖色辅助色用于关键行动点和机会标识，让整体感觉更"有商机"。

```
Amber（琥珀）— 机会、收益、行动号召
  Amber-50:   #FFFBEB   ← 背景
  Amber-400:  #FBBF24   ← 图标/强调
  Amber-500:  #F59E0B   ← 主色（CTA按钮/机会标识）
  Amber-600:  #D97706   ← 深色

Orange（橙色）— 紧迫、限时、热门
  Orange-50:  #FFF7ED   ← 背景
  Orange-500: #F97316   ← 主色（热门标签/限时标识）
  Orange-600: #EA580C   ← 深色
```

**使用规则**：
- 主色 Indigo 用于：导航、品牌标识、常规按钮、链接
- 暖色 Amber 用于：机会商品标识、收益数据、CTA按钮、"今日机会榜"标题
- 暖色 Orange 用于：热门标签、限时标识、爆发品类标记
- 同一组件不超过2种颜色，避免视觉混乱

### 2.9 暗色模式规划（Phase 2）

暗色模式采用"表面层级"系统，而非简单反转颜色：

```
Light → Dark 映射规则：
  页面背景: #F8FAFC → #0F172A (Slate-900)
  卡片背景: #FFFFFF → #1E293B (Slate-800)
  悬浮卡片: #FFFFFF → #334155 (Slate-700)
  边框:     #E2E8F0 → #334155 (Slate-700)
  主文字:   #0F172A → #F8FAFC (Slate-50)
  次要文字: #475569 → #94A3B8 (Slate-400)
  辅助文字: #94A3B8 → #64748B (Slate-500)

侧边栏暗色模式：
  背景: #0F172A (Slate-900) — 比Light模式更深
  文字: #F8FAFC (主) / #94A3B8 (次)
  活跃态: rgba(99,102,241,0.3)

语义色暗色模式：
  Success: #34D399 (更亮)
  Warning: #FBBF24 (更亮)
  Danger:  #F87171 (更亮)
  Info:    #38BDF8 (更亮)

实现方式：
  CSS变量 + data-theme="dark" 属性选择器
  跟随系统偏好 + 手动切换
```

统计卡片中的大数字使用 `Inter` 字体的等宽数字特性：

```css

  font-variant-numeric: tabular-nums;
  font-size: 36px;
  font-weight: 700;
  line-height: 44px;
  letter-spacing: -0.02em;
}
```

## 四、间距系统

### 4.1 基准单位

4px 为基准，所有间距为4的倍数：

```
4px   → 微间距（图标与文字）
8px   → 紧凑间距（标签内padding）
12px  → 小间距（卡片内元素）
16px  → 标准间距（卡片padding）
20px  → 中间距（区域间距）
24px  → 大间距（页面padding）
32px  → 超大间距（区域分隔）
40px  → 页面边距
48px  → 最大间距
```

### 4.2 各场景间距

| 场景 | 间距 |
|-----|------|
| 页面内容区padding | 24px |
| 卡片内padding | 16px-20px |
| 卡片间距 | 16px |
| 统计卡片间距 | 16px |
| 表格行间距 | 12px |
| 按钮组间距 | 8px |
| 标签组间距 | 8px |
| 侧边栏item间距 | 4px |
| 侧边栏分组间距 | 16px |
