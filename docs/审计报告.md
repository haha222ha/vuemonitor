### 7.2 必须修复项（生产部署前）

1. **更换所有默认密钥**：JWT_SECRET、JWT_REFRESH_SECRET、ENCRYPTION_KEY
2. **更换数据库密码**：PostgreSQL 和 Redis 密码
3. **配置 HTTPS**：部署 Nginx 反向代理 + SSL 证书
4. **限制端口暴露**：移除 PostgreSQL/Redis 的宿主机端口映射
5. **配置 CORS 白名单**：限制为生产域名

### 7.3 建议改进项

1. 添加 Nginx 反向代理配置（含 SSL）
2. 使用 Docker Secrets 或 Vault 管理敏感配置
3. 对数据库备份文件启用加密
4. 添加日志脱敏过滤器
5. 锁定生产环境依赖版本
