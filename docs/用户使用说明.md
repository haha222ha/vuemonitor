## 1. 产品简介

XHS365 是一款小红书 AI 选品数据监控与分析工具，帮助电商从业者：

- 🔍 **实时监控**：自动采集小红书商品数据，追踪价格、销量、评分变化
- 📊 **智能分析**：AI 驱动的趋势分析、竞品对比、选品推荐
- ⚡ **即时告警**：价格异动、销量激增等关键事件实时推送
- 👥 **团队协作**：共享监控规则和商品数据，协同选品决策

### 系统组成

| 组件 | 说明 | 适用场景 |
|------|------|----------|
| 桌面客户端 | Electron 桌面应用，支持本地采集和离线使用 | 日常选品、数据采集 |
| Web 管理端 | 浏览器访问，轻量化使用 | 团队协作、移动办公 |
| 服务端 | 后端 API 服务，数据存储和 AI 分析 | 系统支撑 |

## 2. 快速上手

### 2.1 安装客户端

1. 下载安装包（Windows/Mac/Linux）
2. 运行安装程序
3. 首次启动，进入登录页面

### 2.2 注册账号

1. 点击登录页面的「注册」链接
2. 输入邮箱和密码
3. 完成注册后自动登录

### 2.3 激活授权

1. 进入「授权管理」页面
2. 输入授权码
3. 点击「激活」
4. 激活成功后，根据套餐解锁对应功能

### 2.4 开始第一次采集

1. 点击左侧导航「商品」
2. 点击「添加采集」按钮
3. 输入小红书商品链接或关键词
4. 选择采集模式（实时/后台）
5. 点击「开始采集」
6. 等待采集完成，查看商品数据

## 3. 功能详解

### 3.1 数据看板

数据看板是系统的首页，提供关键指标一览：

| 卡片 | 说明 |
|------|------|
| 监控商品数 | 当前监控的商品总数 |
| 今日采集 | 今日采集的数据条数 |
| 告警事件 | 未处理的告警数量 |
| AI 分析 | AI 分析报告数量 |
| 价格趋势 | 商品价格变化趋势图 |
| 销量排行 | 销量增长最快的商品 |
| 分类分布 | 商品分类占比图 |

**自定义看板**：点击右上角「自定义看板」按钮，可拖拽调整卡片顺序、显示/隐藏卡片。

### 3.2 商品管理

#### 商品列表

- 支持按平台、分类、价格区间筛选
- 支持关键词搜索
- 支持列表/卡片视图切换
- 支持批量导出（CSV/JSON）

#### 商品详情

点击商品进入详情页，查看：

| 区域 | 内容 |
|------|------|
| 基本信息 | 商品名称、价格、销量、评分、店铺 |
| 价格趋势 | 历史价格变化折线图 |
| 销量趋势 | 历史销量变化折线图 |
| 排名百分位 | 商品在分类中的排名位置 |
| 品类基准对比 | 与品类平均值的雷达图对比 |
| AI 分析 | AI 生成的商品分析报告 |

#### 商品对比

1. 在商品列表勾选 2-5 个商品
2. 点击「对比」按钮
3. 查看多维度对比表格和雷达图

### 3.3 数据采集

#### 采集模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 实时采集 | Chromium 浏览器实时加载，数据最新 | 需要最新数据的场景 |
| 后台采集 | Playwright 无头浏览器，不干扰工作 | 批量采集、定时采集 |

#### 采集配置

- **并发数**：同时采集的任务数（1-5）
- **采集间隔**：两次采集之间的间隔时间
- **代理设置**：HTTP/SOCKS5 代理
- **Cookie 管理**：登录态管理，支持导入/导出

#### 风控对抗

系统内置风控对抗机制，自动处理：

- User-Agent 轮换
- 请求间隔随机化
- 验证码检测与暂停
- 行为特征模拟
- Cookie 健康检查

当检测到风控风险时，系统会自动暂停采集并通知用户。

### 3.4 监控告警

#### 创建监控规则

1. 进入「监控」页面
2. 点击「新建规则」
3. 选择规则类型：

| 规则类型 | 说明 | 示例 |
|----------|------|------|
| 价格下降 | 商品价格低于阈值 | 价格低于 50 元 |
| 价格上涨 | 商品价格高于阈值 | 价格高于 200 元 |
| 销量激增 | 销量增长率超过阈值 | 7 天销量增长 > 100% |
| 评分下降 | 评分低于阈值 | 评分低于 4.0 |
| 库存变化 | 库存数量变化 | 库存低于 10 |

4. 配置触发条件（阈值、时间窗口等）
5. 设置告警级别（信息/警告/严重）
6. 保存规则

#### 规则模板

系统提供预设规则模板，一键应用：

- 价格监控模板
- 销量监控模板
- 综合监控模板

### 3.5 AI 分析

#### 趋势分析

1. 选择商品
2. 点击「AI 分析」
3. 选择分析类型：趋势预测/竞品对比/选品建议
4. 查看分析报告

#### 报告生成

1. 进入「AI 分析」页面
2. 选择商品和报告模板
3. 点击「生成报告」
4. 报告支持 PDF 导出

#### 报告模板

| 模板 | 说明 |
|------|------|
| 商品分析报告 | 单品深度分析 |
| 趋势预测报告 | 未来趋势预测 |
| 竞品对比报告 | 多商品对比分析 |
| 选品建议报告 | AI 选品推荐 |

### 3.6 通知中心

通知中心汇总所有系统通知：

| 通知类型 | 说明 |
|----------|------|
| 告警通知 | 监控规则触发的告警 |
| 采集通知 | 采集任务完成/失败 |
| 同步通知 | 数据同步状态 |
| 系统通知 | 更新/授权等系统消息 |

操作：
- 点击通知查看详情
- 勾选通知后可批量标记已读/删除
- 支持按类型筛选

### 3.7 数据同步

#### 同步模式

| 模式 | 说明 |
|------|------|
| 自动同步 | 定时自动同步本地数据到云端 |
| 手动同步 | 点击「立即同步」触发 |
| 离线模式 | 网络断开时操作入队，恢复后自动同步 |

#### 冲突解决

当本地和云端数据冲突时：

1. 系统自动检测冲突
2. 在通知中心提示冲突
3. 用户选择解决方案：
   - 保留本地版本
   - 保留云端版本
   - 合并版本

### 3.8 团队协作（Web 端）

#### 创建团队

1. 进入 Web 端「团队协作」页面
2. 点击「创建团队」
3. 输入团队名称和描述

#### 邀请成员

1. 在团队页面点击「邀请成员」
2. 输入成员邮箱
3. 选择角色（管理员/成员）
4. 发送邀请

#### 资源共享

- 共享监控规则
- 共享商品数据
- 共享 AI 报告

## 7. 常见问题

### Q1: 采集速度慢怎么办？

**可能原因和解决方案**：

1. **并发数过低**：在设置中提高并发数（建议 3-5）
2. **网络延迟**：检查网络连接，考虑使用代理
3. **风控限制**：系统自动降低采集速度以规避风控，等待恢复
4. **目标网站响应慢**：增加采集间隔时间

### Q2: 采集被风控拦截怎么办？

系统会自动检测风控并暂停采集。处理方式：

1. 等待一段时间后恢复采集
2. 更换代理 IP
3. 更新 Cookie（重新登录）
4. 降低采集频率

### Q3: 数据同步冲突如何处理？

1. 进入「设置 → 冲突管理」
2. 查看冲突列表
3. 选择解决方案（本地优先/云端优先/合并）
4. 确认解决

### Q4: AI 分析报告生成失败？

**可能原因**：

1. **API Key 未配置**：联系管理员配置 DeepSeek API Key
2. **配额用尽**：检查当前套餐的 AI 调用配额
3. **网络问题**：检查与服务端的连接

### Q5: 授权码激活失败？

**可能原因**：

1. 授权码格式错误：检查是否完整复制
2. 授权码已使用：一个授权码只能激活一台设备
3. 授权码已过期：联系销售获取新授权码
4. 网络问题：检查与服务端的连接

### Q6: 客户端自动更新失败？

1. 检查网络连接
2. 检查是否有管理员权限
3. 手动下载最新版本安装
4. 联系技术支持

### Q7: 离线模式下数据会丢失吗？

不会。离线模式下的所有操作都会被记录到本地队列，网络恢复后自动同步到云端。

### Q8: 如何更换绑定的设备？

1. 在当前设备上「授权管理」页面点击「解绑」
2. 在新设备上重新输入授权码激活
3. 如无法访问原设备，联系管理员远程解绑

### Q9: 如何导出所有数据？

1. 客户端：「设置 → 数据管理 → 导出全部数据」
2. Web 端：各列表页面支持 CSV/JSON 导出
3. API：调用 `/api/v1/products/export` 接口

### Q10: 团队协作如何收费？

团队协作功能包含在 Pro 及以上套餐中：

| 套餐 | 团队人数 | 共享规则 | 共享商品 |
|------|----------|----------|----------|
| Free | 不支持 | — | — |
| Pro | 5 人 | 10 条 | 50 个 |
| Premium | 20 人 | 50 条 | 200 个 |
| Enterprise | 不限 | 不限 | 不限 |
